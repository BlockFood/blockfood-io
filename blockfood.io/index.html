BlockFood

Give the sharing economy back to the people with Smart Contracts and Ethereum

What is BlockFood?

BlockFood is an open platform created to connect customers to restaurants with a fair distribution model of revenues.

It is based on Smart Contracts and Ethereum.

BlockFood is a non-profit project.

BlockFood is the first project from the Open Sharing Economy foundation.

Key characteristics

Decentralized
Users are empowered. They make the platform grow and they reap the benefits.
Fair
Revenues are distributed fairly between all actors.
BlockFood is a non-profit project.
Open
The source code is available on open source platforms.
The platform is open to third-party developers.

How does it work?

The platform uses smart contracts to handle the payment and the progression of the orders placed by customers.
